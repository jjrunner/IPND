Important Concepts

How the Web Works

There are 4 important ideas to keep in mind with regard to the web:

  • Servers are computers that host the files that make up the web.

  • Browsers are programs that run on our computer display files found on the web. Common browsers include Chrome, Firefox, and Internet Explorer.

  • The Internet is the world's largest computer network. Company's have built search engines to help us parse through the information we want on the Internet. The largest of these companies is Google.

  • HTTP is the main protocol of the web. It is a way for the servers and browsers to communicate with one another. You can think of the Internet as the tubes that connect everything, and HTTP as the way that servers and clients chat with one another across those tubes.

Basics of HTML

HTML stands for HyperText Markup Language. This is the main set of text that is used to communicate on the Web. In this lesson, we focused on a few of the most common tags used in HTML. These included:

  • <b> This is the bold tag.

  • <em> This is the italic tag.

  • <b> This is the bold tag.

  • <img> This is an tag, which uses a src attribute.

  • <a> This is a tag that allows us link pages together. For example, I can link to a page of a www.animalplanet.com/pets/dogs/ using this tag.

Beyond the tags above, we looked at the difference between Inline and Block elements. That is, some of these tags take place right within a line, and others create a box around our text. We didn't get into the details of this, but I am sure we will talk about why we might do one of these versus another in future lessons.

Notice that some tags do not require an ending tag. These are known as void tags.
